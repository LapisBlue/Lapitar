Warning: Lapitar is still in development and shouldn't be used in production yet. It will be down from time to time when we fix bugs or update it with new features.

Lapitar is a new open source Minecraft avatar service, providing 2D and 3D avatars with a simple API.

Note: Although it is not required, we highly recommend getting avatars by UUIDs instead of the names in bigger applications. For us, getting an avatar by UUID is faster and more reliable than by name. You can make your application more future-proof by converting everything to UUIDs as soon as possible.

Faces

A simple, flat avatar with only the face of the player's skin. Use helm instead of face to add the head overlay additionally.

https://lapitar.lapis.blue/face/[Size]/<UUID|Name>[.png]
https://lapitar.lapis.blue/helm/[Size]/<UUID|Name>[.png]

Example:

https://lapitar.lapis.blue/face/Aesen_.png
https://lapitar.lapis.blue/helm/256/93a09a981fbb46da85a6f0bb1465dc53

Heads

A perspective avatar with the head of the player's skin.

https://lapitar.lapis.blue/head/[Size]/<UUID|Name>[.png]

Example:

https://lapitar.lapis.blue/head/Aesen_.png
https://lapitar.lapis.blue/head/256/93a09a981fbb46da85a6f0bb1465dc53

Portraits

A perspective avatar with a portrait of the player's skin.

https://lapitar.lapis.blue/portrait/[Size]/<UUID|Name>[.png]

Example:

https://lapitar.lapis.blue/portrait/Aesen_.png
https://lapitar.lapis.blue/portrait/256/93a09a981fbb46da85a6f0bb1465dc53

Players

A full render of the player's skin.

https://lapitar.lapis.blue/player/[Size]/<UUID|Name>[.png]

Example:

https://lapitar.lapis.blue/player/Aesen_.png
https://lapitar.lapis.blue/player/256/93a09a981fbb46da85a6f0bb1465dc53

Skins

The raw skin of the player. This will return the player's skin as-is, with no modifications.

https://lapitar.lapis.blue/skin/<UUID|Name>[.png]

Example:

https://lapitar.lapis.blue/skin/Aesen_.png
https://lapitar.lapis.blue/skin/93a09a981fbb46da85a6f0bb1465dc53
